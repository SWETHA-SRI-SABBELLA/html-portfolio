# StudentVerse MVP

## Problem
Students need a simple system that links stress, study plans and sustainability.

## Solution
StudentVerse: Typing Stress Test -> Adaptive Study Scheduler -> Carbon Calculator.

## How to run (open in browser)
Open index.html in the `www` folder and use the links.
